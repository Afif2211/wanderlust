# Wanderlust 🌍✨

Wanderlust is a modern Airbnb-inspired full-stack web application where users can explore unique destinations, create property listings, upload images, leave reviews, and discover amazing places around the world.

Built with a focus on clean UI, responsive design, authentication, cloud storage, and interactive maps.

---

## 🚀 Live Demo

🔗 [View Live](https://wanderlust-psgr.onrender.com)

---

## 📌 Features

### 🔐 Authentication & Authorization
- Secure Signup/Login system using Passport.js
- Session-based authentication
- Protected routes for authorized users only

### 🏡 Listings
- Create, edit, and delete property listings
- Upload property images via Cloudinary
- Beautiful responsive listing cards

### ⭐ Reviews & Ratings
- Users can add and delete reviews
- Star rating system (1–5) for listings

### 🗺️ Maps Integration
- Interactive maps using Leaflet + OpenStreetMap
- Geocoding via Mapbox API for accurate location markers
- Real-time location visualization on listing pages

### ☁️ Cloud Storage
- Cloudinary integration for image uploads
- Optimized image handling and delivery

### 📱 Responsive UI
- Mobile-friendly modern design
- Custom navbar with search functionality
- Clean and intuitive user experience

## ⚙️ Installation

**1. Clone the Repository**
```bash
git clone https://github.com/Afif2211/wanderlust.git
```

**2. Move into Project Folder**
```bash
cd wanderlust
```

**3. Install Dependencies**
```bash
npm install
```

**4. Create `.env` File**

Add the following environment variables:
```env
ATLASDB_URL=your_mongodb_connection_string

CLOUD_NAME=your_cloudinary_name
CLOUD_API_KEY=your_cloudinary_key
CLOUD_API_SECRET=your_cloudinary_secret

MAP_TOKEN=your_mapbox_token

SECRET=your_session_secret
```

**5. Start the Application**
```bash
node app.js
```
